Movie API

The Movie API will allow the user to different information from movies, such as genre and director, and create lists of their favorites. The will also be able t sign up and update their personal information.

Features

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it/s featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., “Drama”)
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info (username)
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister

API Endpoints

Goal URL HTTP Method Response body data format
Get a list of all movies /movies GET A JSON object holding data about all the movies
Get a single movie by title /movies/:title GET A JSON object holding data about a single movie
Get Genre by Name /movies/genre/:genreName GET A JSON object holding data about the genre of a movie
Get director data by name /movies/director/:directorName GET A JSON object holding data about the data of a director
Allow new users to Register /users POST A JSON object holding data about the user that was added
Allow new users update their user info /users/:id PUT A JSON object holding data about the updated information
Adds movie to favorite list /users/:id/:movieTitle POST A text message indicating whether the Movie was added
Delete movie from favorite list /users/:id/:movieTitle DELETE A text message indicating whether the Movie was removed
Allow existing users to deregister /users/:id DELETE A text message indicating whether the User has deregistered
